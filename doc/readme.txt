examination paper generator

功能：
1、能根据模版和定义自动生成题目
模版：
#<a>和#<b>合计有#<c>#<d>，#<a>是#<b>的#<e>倍，他们各有多少#<d>？
定义：
#<a> = r(person)  //人
#<b> = r(person) 
#<c> = r(100)
#<d> = r(object) //物
#<e> = r(10)     //10以内的随机数
matrix = {1,1,#<c>},{1,-#<e>,0}

可以生成题目为：
小明和小王合计有20本书，小明是小王的3倍，他们各有多少本书？
张三和李四合计有24支笔，张三是李四的5倍，他们各有多少支笔？
......
